# pySGE
 Simple Graphics Engine (SGE) is a graphics engine written in python that provides an easy way to compose 3D scenes in realtime using a variety of graphics pipelines and APIs.
 
 Can be paired with pygame and a physics engine to create 3D games.

## Testing
### Install dependencies
```pip install pyopengl pyopengl-accelerate pyopengltk pywavefront pygame numpy```

### Run example
```python example.py```
